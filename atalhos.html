Atalhos do Windows

Ctrl + C

Copiar o item selecionado.

Ctrl + V

Colar o item copiado.

Ctrl + X

Cortar o item selecionado.

Ctrl + Z

Desfazer a última ação.

Ctrl + y

Refazer a última ação desfeita.

Ctrl + A

Selecionar todos os itens.

Ctrl + s

Salvar o arquivo atual.

Ctrl + shift + esc

Abrir o Gerenciador de Tarefas.

alt + Tab

Alternar entre janelas abertas.

Alt + prtscn

Abrir o Explorador de Arquivos.

Windows + L

Windows + R

Abrir a caixa de diálogo Executar.

Windows + S

Abrir a barra de pesquisa.

Windows + X

Abrir o menu de acesso rápido.

Windows + I

Abrir as configurações do Windows.

Windows + setas

Configurar a projeção da tela.

Windows + Tab

Abrir a visão de tarefas.

Windows + M

Minimizar todas as janelas abertas.

Windows + Shift + M

Restaurar todas as janelas minimizadas.

Windows + PrtScn

Capturar a tela atual.

Windows + Shift + S

Salvar a tela atual.
